1. No terminal do Linux, para iniciar o ambiente virtual, faça "source env/bin/activate" dentro da pasta do projeto
2. Para rodar a API, "python3 manage.py runserver"
3. No navegador, acesse "http://120.0.0.1:8000/" para a página inicial
 \_3.1. Digite o IP e aperte Enter para pesquisar ou pressione o botão "Pesquisar"
4. Para visualizar os JSONs das pesquisas anteriores, acesse "http://120.0.0.1:8000/previous/"
